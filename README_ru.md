# Map Generators — расширение для SillyTavern

Встраивает процедурные генераторы карт [Watabou](https://watabou.itch.io/) прямо в
SillyTavern, ведёт **библиотеку карт для каждого чата** и передаёт ИИ текстовое
описание каждой карты, чтобы он понимал и запоминал места в вашей истории.

## Установка

В SillyTavern откройте **Extensions → Install extension** и вставьте ссылку:

```
https://github.com/XaYS-101/Map-generators-Silly-Tavern
```

## Возможности

- **Живой генератор в popup** — Город (Medieval Fantasy City), Деревня,
  One Page Dungeon, Perilous Shores, Dwellings, Таверны, Urban Places.
  Интерактивный генератор открывается в модальном окне (iframe нельзя держать
  внутри сообщения чата — ST его вырезает).
- **Сохранение/восстановление** — состояние карты целиком закодировано в её URL
  (`?seed=…&tags=…`) и хранится в метаданных чата, поэтому переживает перезаход.
  Сколько угодно карт, в том числе несколько из одного генератора.
- **Заставить ИИ запомнить карту** — для каждой карты выбор:
  - **Лорбук чата** — запись World Info, привязанная к чату, срабатывает по
    названию места (экономит токены).
  - **Инжект в промпт** — всегда в контексте (как Author's Note), через
    `setExtensionPrompt`.
- **Три источника описания** (всё редактируемо):
  1. **Импорт JSON** — экспортируйте JSON из генератора Города / Perilous Shores
     и загрузите файл; он разбирается в текстовое описание.
  2. **ИИ-vision** — задайте PNG-превью и пусть мультимодальная модель его опишет.
  3. **Вручную**.
- **Превью в чат** — по желанию прикрепить PNG к последнему сообщению (заодно
  становится видно vision-модели).
- Интерфейс EN / RU, адаптирован под телефон.

## Использование

1. Откройте библиотеку: команда **`/map`** или кнопка **Открыть библиотеку карт**
   в `Extensions → Map Generators`.
2. **+ Новая карта** → выберите генератор → покрутите сид/размер/теги в окне →
   **Сохранить эту карту**.
3. В редакторе впишите/импортируйте/сгенерируйте **описание**, затем выберите
   режим **памяти ИИ**. Именно это читает модель.

> Про cross-origin: расширение не может прочитать состояние из iframe генератора,
> поэтому картой управляют его собственные поля сид/размер/теги. Если вы крутили
> карту кнопками самого генератора — вставьте полученный URL в поле
> *«…или вставьте URL карты»*.

## Авторство и лицензия

Карты создаются инструментами **Watabou** на `watabou.github.io`. Готовые карты
свободны для использования, копирования и изменения (авторство приветствуется).
Расширение лишь **встраивает публичные генераторы** и не распространяет их код.
Указывайте авторство Watabou и по возможности поддержите автора на
[itch.io](https://watabou.itch.io/).

## Примечания

- Базовые URL для `taverns` и `urban` стоит сверить с актуальным хостом «Run game»
  на itch.io при первом использовании; остальные проверены.
- itch.io-инструменты (таверны, urban, иконки, tiny-pubs, constellations, сигилы,
  histomap) открываются в новой вкладке — у них нет seed/URL-состояния.
- Для ИИ-vision нужна настроенная мультимодальная модель.
